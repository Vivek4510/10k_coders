Frontend Technology
Front-end technologies serve as the foundation and building blocks of a website, defining its user interface and shaping the overall user experience. Front-end enables you to interact with the web and its actions. In modern web development, an array of tools, frameworks, and practices contribute to creating engaging and responsive websites.
HTML
  • HTML stands for Hyper Text Markup Language.
  • It is used to create Web pages.
  • It is used to create web page structure.
  • Divided into Static and Dynamic pages.
Heading Tag
  • It is used to denote headings in a HTML page
  • There are 6 heading tags from h1 to h6
  • h1 gives the largest heading
  • h6 gives the smallest heading
Paragraph Tag
  • It is denoted by < p > tag
  • It is used to wrap the text according to the device's screen size
  • It eliminates unecessary space in between the words
Anchor Tag
  1. It is represented with < a > tag.
  2. It has attributes like href and target.
  3. href is used to create hyper links for web pages. It stands for hyper reference.
  4. target are of 4 types
    1. Self
    2. Blank
    3. Parent
    4. Top
Tables
  1. It is used to represent data in a tabular form.
  2. It is denoted by < table > tag.
  3. Table is divided into rows and columns. The sub tags of the table are tr,td,th.
  4. th stands for Table heading.
  5. tr stands for Table row.
  6. td stands for Table data.
  7. The attributes used in table tag are :
    1. Border : It is used to specify the borders of the table.
    2. Caption : It gives a caption for the table.
    3. Cell Padding : It is used to create space around the content in a cell.
    4. Cell Spacing : Create space between two cells.
    5. bgcolor : Add background color to the table.
Lists
  1. There are three types of lists:
    1. Ordered List
    2. Unordered List
    3. Description List
  2. Ordered List : It is used to specify list items in a sequence manner.
  3. The attributes in Ordered list are:
    1. Type
    2. Start
    3. Reversed
  4. Unordered List : It is used to specify list items in a particular order.
  5. Description List : It is used to create description items.
  6. It has subtags like description term and description data, they are represented with < dt > and < dd > respectively.
